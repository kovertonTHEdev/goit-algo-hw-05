# Python Домашні Завдання

Домашні завдання з Python відповідно до технічного завдання.

---

## Структура репозиторію

### hw_01 — Замикання та кешування (Fibonacci)
- `hw_01.py` — реалізація функції caching_fibonacci

Реалізовано:
- Замикання (closure)
- Рекурсія
- Кешування через словник

Запуск:
python hw_01.py


---

### hw_02 — Генератор чисел і підрахунок суми
- `hw_02.py` — функції generator_numbers та sum_profit

Реалізовано:
- Генератор (yield)
- Пошук дійсних чисел через regex
- Типізація з Callable
- Підрахунок загальної суми

Запуск:
python hw_02.py


---

### hw_03 — CLI-аналізатор лог-файлів
- `hw_03.py` — аналіз лог-файлів
- `log_file.txt` — приклад лог-файлу

Можливості:
- Прийом шляху до файлу через CLI
- Підрахунок записів за рівнями (INFO, DEBUG, ERROR, WARNING)
- Фільтрація за рівнем логування
- Фіксований порядок виводу
- Обробка помилок

Запуск:
python hw_03.py log_file.txt
python hw_03.py log_file.txt ERROR (або інший тип помилок)


---

### hw_04 — Консольний бот
- `hw_04.py` — бот для роботи з контактами

Команди:
- add
- change
- phone
- all
- hello
- exit

Особливості:
- Декоратор input_error
- Обробка ValueError, IndexError, KeyError
- Програма не завершується при помилках

Запуск:

python hw_04.py
